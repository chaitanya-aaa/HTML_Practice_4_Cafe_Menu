# ☕ Chateau Cafe Menu Webpage

## 📌 Project Overview

This project demonstrates the creation of a **simple Cafe Menu webpage using HTML**. The webpage displays a list of coffee and dessert items along with their prices in a structured menu format.

The main objective of this project is to practice **basic HTML structure, text formatting, images, sections, and hyperlinks** while designing a simple menu interface.

This project highlights fundamental HTML concepts including **headings, images, sections, horizontal lines, hyperlinks, and page layout**.

---

## 🎯 Problem Statement

To design a simple cafe menu webpage that displays coffee and dessert items along with their prices in a clear and visually organized format.

---

## 🛠️ Project Modules Covered

### 1️⃣ HTML Document Structure
- Created the base webpage using `html`, `head`, and `body`.
- Included meta tags for character encoding and responsive display.

### 2️⃣ Header Section
- Displayed the cafe name **CHATEAU CAFE** using a heading.
- Added establishment year information.

### 3️⃣ Coffee Menu Section
- Listed different coffee items and their prices.
- Included an image representing coffee.

### 4️⃣ Dessert Menu Section
- Listed dessert items and their prices.
- Displayed an image representing desserts.

### 5️⃣ Content Organization
- Used sections and horizontal lines to separate menu categories.

### 6️⃣ Footer Section
- Added contact details, copyright information, and a website link.

## 🌟 Key Features
